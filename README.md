# Date & Time Range Picker (Frontend Engineering Assignment)

This project is an implementation of a **timezone-aware, accessible Date & Time Range Picker** built entirely from scratch as part of a frontend internship assignment.

The goal of this project is to demonstrate understanding of:
- React state modeling
- Calendar and date logic
- Accessible, keyboard-first UI design
- Constraint handling and validation
- Component-driven development using Storybook
